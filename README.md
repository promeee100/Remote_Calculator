## **Python Socket Programming Based Project**

#  Remote Calculator
This is a lightweight, fast, and visually captivating calculator application designed with a Deep Purple Accent and Hipii Blue color palette. The UI is inspired by modern material design trends—smooth corners, soft shadows, and a premium aesthetic.

## Installation

**Clone the repository**

git clone - https://github.com/promeee100/Remote_Calculator


**Navigate to the project folder**

cd python-Remote Calculator


**Install dependencies (if any)**

pip install -r requirements.txt

## Usage

**Run the project using:**

python server.py

python client.py


**The server will start on:**

http://127.0.0.1:6000

## Features

✔ Modern UI with deep purple and hipii-blue dual-tone theme

✔ Smooth and responsive button interactions

✔ Large, readable display area

✔ Clean layout using GridView for button arrangement

✔ Support for addition, subtraction, multiplication, and division

✔ Error-free expression handling

✔ Lightweight & fast performance

## Design Philosophy

1. Minimal & Clean Layout

2. High Contrast Color Palette (Deep Purple × Hipii Blue)

3. Smooth Rounded Corners (16–24 px)

4. Consistent Typography

5. Professional button elevations & spacing

This combination creates a premium, glassy, modern calculator experience
